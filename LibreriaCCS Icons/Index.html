Documentación Libre

Esta librería ha sido creada por ANIMA Studio con el propósito de apoyar a creadores independientes, diseñadores y desarrolladores que buscan herramientas accesibles y funcionales para sus proyectos. Es de uso libre, siempre y cuando se respete su propósito: fomentar la creatividad, la colaboración y el crecimiento tecnológico desde una comunidad independiente. Para utilizar los íconos y estilos de esta librería, simplemente vincula la siguiente API en tu documento HTML:

Vincula esta API en tu documento HTML: 
<link rel="stylesheet" href="https://animastudio-ignacioacevedo-librerias.netlify.app/LibreriaIcons.CSS">
¡Copiado!

Con esta línea, tendrás acceso inmediato a todos los estilos e íconos diseñados por ANIMA Studio . Estamos desarrollando mejoras.
